# PinolApp v10

PinolApp es una plataforma de delivery similar a Uber Eats.

## Tecnologías

- Flutter (Apps móviles)
- Node.js + Express (Backend)
- PostgreSQL (Base de datos)
- Next.js (Panel web)

## Estructura

apps/
web/
backend/
database/

## Instalación

Backend:

cd backend/node_api
npm install
npm run dev

Admin panel:

cd web/admin_nextjs
npm install
npm run dev
